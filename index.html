What is it?

Awesome Inc U is a code school. Through hands-on, in-person classes, we help people learn how to create software. It’s never too early or too late to learn how to code. You don’t have to be a genius to learn to code, you just need to be determined. From musicians, to athletes, to billionaire founders of tech companies, lots of successful people want YOU to learn to code.
